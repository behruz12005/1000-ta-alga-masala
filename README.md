# 1000-ta-alga-masala
“1000-alga-masala” loyihasining asosiy maqsadi — dasturchilarning algoritmik fikrlash qobiliyatini muntazam ravishda oshirish, muammolarni tahlil qilish va samarali yechim topish ko‘nikmasini mustahkamlashdir. Ushbu loyiha orqali biz bir vaqtning o‘zida ham algoritm asoslarini chuqur o‘rganamiz, ham real dasturchilik tajribasi olish .
